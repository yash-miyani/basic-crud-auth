# Next.js CRUD Auth App

Simple project using:

- Next.js (App Router)
- Tailwind CSS v4
- Light / Dark Mode
- Login & Register
- Product CRUD
- Axios API
- Lucide Icons

---

## Project Setup

### 1. Create project

```bash
npx create-next-app@latest client
cd client
```

### 1. Install packages

npm install axios next-themes lucide-react

### 3. Folder Structure
