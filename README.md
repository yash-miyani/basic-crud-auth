# BASIC-CRUD-AUTH

This is a **full-stack CRUD application with authentication**, built with a Next client and Node.js/Express backend.

## Project Structure

│
├─ client/ # React frontend
├─ server/ # Node.js backend with MongoDB
├─ .gitignore
└─ README.md

## Features

- User registration and login with email/password
- CRUD operations for products
- Separate client and server directories
- Pagination and search functionality
- Error handling

## Setup

### 1. Backend

```bash
cd server
npm install
npm run dev
```

### 2. Frontend

```bash
cd client
npm install
npm start

```
